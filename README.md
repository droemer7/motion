

# Motion

A collection of mobile robot trajectory generation algorithms for path and velocity profiles, integrated with ROS.

**Author:** Dane Roemer

</br>

There is nothing here right now, other than scratch test code for testing the optimization library [CppNumericalSolvers (fork as ROS package)](https://github.com/droemer7/CppNumericalSolvers) (which this repo will utilize).

</br>

## References
